# Semana 07: UI Modernas con Tailwind y Material Design

## 🎯 Objetivo
Agilizar el proceso de desarrollo Frontend usando frameworks de CSS basados en utilidades o librerías de componentes completas.

## 🛠️ Stack Tecnológico
- Tailwind CSS
- Material UI (MUI) o Shadcn/ui

## 📋 Requisitos del Proyecto
1. Convertir el proyecto del Carrito de Compras para que sea 100% responsivo (Móvil, Tablet, Desktop) utilizando exclusivamente **Tailwind CSS**.
2. Evitar escribir CSS estándar o CSS Modules.
3. En una nueva vista, integrar una librería de componentes basada en Material Design (MUI) para crear un formulario completo (Validación de tarjeta de crédito estática, dirección, nombres) aprovechando sus TextFields preconfigurados.
4. Explorar variantes de componentes y *Utility classes*.

## ✅ Entregables
- Proyecto escalable visualmente.
- Aplicación de diseño centrado en el usuario (UX).
