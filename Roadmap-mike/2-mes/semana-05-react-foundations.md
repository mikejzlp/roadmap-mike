# Semana 05: React Foundations (Componentes y Hooks)

## 🎯 Objetivo
Aprender la base del entorno React moderno: el Virtual DOM, Vite, Funciones y Hooks básicos.

## 🛠️ Stack Tecnológico
- ReactJS
- Vite
- TypeScript
- CSS Modules

## 📋 Requisitos del Proyecto
1. Crear un proyecto nuevo con Vite y React-TS.
2. Desarrollar un "Gestor de Tareas" (To-Do list) dividiendo la UI en Componentes funcionales pequeños.
3. Usar el hook `useState` para crear, editar, marcar como completadas y eliminar tareas.
4. Usar `useEffect` para guardar las tareas localmente en el `localStorage` del navegador.
5. Empezar a utilizar tipado duro de TypeScript para las *Props* de los componentes (`interface Props`).

## ✅ Entregables
- Proyecto desplegado de manera estática (Vercel, Netlify o GitHub Pages).
- Estructura de componentes clara y reutilizable.
