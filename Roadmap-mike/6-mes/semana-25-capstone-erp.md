# Semana 25: Proyecto Capstone Final - ERP B2B Corporativo

## 🎯 Objetivo
Este es tu Masterpiece (Obra Maestra) que condensa el propósito del "Full Stack". Demostrar dominio en todo el ciclo de visa de software, construyendo el mismo ERP y ecosistema que mencionaban en ofertas como la de "Elektro3".

## 🛠️ Stack Tecnológico
- **Cloud/Infra:** AWS EC2 + GitHub Actions CI/CD + Docker/K8s
- **Backend:** Node.js + Express (Clean Architecture) + PostgreSQL (Prisma ORM) + JWT Auth
- **Frontend Web:** React 18+ con Tailwind, Zustand o React Query y Testing (Jest/Cypress).
- **Mobile:** React Native (Sync & APIS).

## 📋 Requisitos del Proyecto
1. **Modelado (Backend):** Crear roles de "Avanzados" o "Administradores", con permisos B2B limitados (una empresa solo ve datos de su empresa `tenantId`).
2. **Interfaz B2B Web:** Tablas masivas de pedidos/inventario usando React y paginado/ordenamiento desde Backend.
3. **App Móvil de Representante (Mobile):** App de React Native que los vendedores usarían offline o consumiendo el backend para verificar stock.
4. **Despliegue (DevOps):** Todo esto automatizado al hacer Push. Servido al mundo con CI/CD, SSL y Dockerización.

## ✅ Entregables
- Un sistema interconectado, testeado, desplegado usando DevOps de alto nivel.
- Repositorios impecables con `README.md` robustos, demostrando no solo código sino habilidades de arquitectura y liderazgo.
