# Semana 01: API REST Node.js & Express

## 🎯 Objetivo
Configurar un servidor backend robusto utilizando Node.js, Express y TypeScript. Diseñar una API RESTful básica para entender el enrutamiento y los middlewares.

## 🛠️ Stack Tecnológico
- Node.js & Express
- TypeScript
- API RESTful

## 📋 Requisitos del Proyecto
1. Inicializar un proyecto con `npm init -y` y configurar TypeScript (`tsc --init`).
2. Levantar un servidor Express en el puerto 3000.
3. Crear un router con las rutas CRUD (GET, POST, PUT, DELETE) para un recurso "Usuarios" (usando un array en memoria momentáneamente).
4. Implementar al menos 2 middlewares: uno para loguear las peticiones entrantes y otro para el manejo global de errores.

## ✅ Entregables
- Repositorio en GitHub con el código fuente.
- Endpoints probados y documentados usando Postman.
- Configuración de `scripts` en `package.json` para modo desarrollo (`ts-node-dev`).
