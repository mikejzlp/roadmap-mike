# Semana 14: React Native Basics (Vistas Nativas)

## 🎯 Objetivo
Traducir los conocimientos de React DOM (Web) al ecosistema Mobile de React Native usando Expo CLI.

## 🛠️ Stack Tecnológico
- React Native
- Expo CLI
- TypeScript
- Flexbox

## 📋 Requisitos del Proyecto
1. Instalar Expo CLI e iniciar un proyecto nuevo `npx create-expo-app@latest`.
2. Crear un clon visual de la pantalla principal de WhatsApp o Spotify.
3. Comprender las diferencias fundamentales: usar `<View>` en lugar de `<div>`, y `<Text>` en lugar de `<p>`.
4. Estilizar las vistas utilizando la prop `style` (StyleSheet.create) o librerías externas modernas.
5. Aprender a renderizar listas infinitas eficientes utilizando `FlatList` o la nueva `FlashList`.

## ✅ Entregables
- Aplicación corriendo en tu simulador iOS/Android o en tu dispositivo físico escaneando el código QR de Expo Go.
