# Semana 15: Navegación en Aplicaciones Móviles

## 🎯 Objetivo
Orquestar rutas y transiciones en aplicaciones móviles utilizando el estándar de la industria.

## 🛠️ Stack Tecnológico
- React Native
- React Navigation V6 (o Expo Router)

## 📋 Requisitos del Proyecto
1. Añadir el paquete `@react-navigation/native` al proyecto.
2. Construir una jerarquía de Navegación "Stack" (Apilar pantallas) desde un Login hacia el Home de la app.
3. Añadir una navegación "Tab" (Barra inferior móvil) en la vista Home (con pestañas para Inicio, Buscar, Perfil).
4. Aprender a pasar parámetros dinámicos entre rutas Móviles (`navigation.navigate('Perfil', { usuarioId: 1 })`).
5. Crear y estilar *Headers* personalizados.

## ✅ Entregables
- Una app Multi-Pantalla con transiciones fluidas de sistema operativo nativo (swipe to go back).
