# Semana 11: E-Commerce Architecture (PERN Stack)

## 🎯 Objetivo
Desplegar una aplicación comercial más compleja utilizando el stack PERN (PostgreSQL + Express + React + Node).

## 🛠️ Stack Tecnológico
- Frontend: ReactJS + Material UI
- Backend: Node.js + PostgreSQL + Prisma ORM
- Pasarela de pagos (Opcional - Stripe Test)

## 📋 Requisitos del Proyecto
1. Bases de datos estricta: Usuarios (Users), Productos (Products), Órdenes (Orders) y Detalle_Órdenes (OrderItems).
2. Carrito de compras que procese el Checkout.
3. Crear un flujo "Transaccional" en la base de datos SQL desde Node (para restar stock solo si se crea el registro completo).
4. Mostrar catálogo de productos en React consumiendo los datos relacionales de Postgres.

## ✅ Entregables
- E-Commerce simulado transaccionando datos en bases relacionales (SQL).
