# Semana 09: Dashboard B2B (MERN Stack)

## 🎯 Objetivo
Unir los conocimientos de Backend (Mes 1) y Frontend (Mes 2) para crear un flujo completo (MERN: MongoDB + Express + React + Node).

## 🛠️ Stack Tecnológico
- Frontend: ReactJS + Tailwind + React Query + Zustand
- Backend: Node.js + Express + MongoDB
- Testing Inicial

## 📋 Requisitos del Proyecto
1. Crear un monolito (o dos repositorios separados) para un CMS / Panel de Control B2B para administrar empleados y gastos.
2. Endpoint `/api/employees` para listar con paginación en Node.js, almacenado en MongoDB.
3. Consumir este endpoint desde React, mostrando los datos en una tabla rica (React Table o DataGrid MUI).
4. El Frontend debe enviar el Bearer Token en las cabeceras generadas en el login hacia la API protegida.

## ✅ Entregables
- Proyecto "Full Stack" operativo de extremo a extremo.
- Manejo correcto de errores cruzados (ej. CORS configurado adecuadamente).
