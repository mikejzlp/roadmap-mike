## 🎯 Objetivo

Construir un portafolio de **26 proyectos** que consolide todas las tecnologías del currículum, formando un perfil **Full Stack Developer** y **Mobile**, con fuertes bases en **DevOps**:

| Tecnología        | Nivel requerido     | Meses donde se practica |
| ----------------- | ------------------- | ----------------------- |
| **Node.js/Express**| Backend principal   | Mes 1, Mes 3            |
| **MongoDB**       | Base de datos NoSQL | Mes 1, Mes 3            |
| **PostgreSQL**    | Base de datos SQL   | Mes 1, Mes 3            |
| **ReactJS**       | Frontend principal  | Mes 2, Mes 3            |
| **React Native**  | Desarrollo Móvil    | Mes 4                   |
| **Docker**        | Contenedores        | Mes 5, Mes 6            |
| **K8s & AWS**     | Orquestación/Cloud  | Mes 6                   |
| **GitHub Actions**| CI/CD pipelines     | Mes 5, Mes 6            |
| **Tailwind & MUI**| UI/UX               | Mes 2, Mes 4            |
| **TypeScript**    | Lenguaje transversal| Todos los meses         |

## 🏗️ Arquitectura del aprendizaje

```
MES 1: Backend Foundations
    Node.js + Express + MongoDB + PostgreSQL
    ↓ Base sólida de backend de datos

MES 2: Frontend & UI
    React + Tailwind CSS + Manejo del Estado
    ↓ Interfaces avanzadas y modernas

MES 3: Full Stack Integration
    Unión del MERN/PERN + Testing + Arquitectura
    ↓ Desarrollo de productos end-to-end

MES 4: Mobile App Development
    React Native + React Navigation + Android
    ↓ Salto al ecosistema móvil y cross-platform

MES 5: DevOps & Contenedores
    Linux + Docker + Docker Compose + CI/CD
    ↓ Operaciones e Integración Continua

MES 6: Cloud & Arquitectura
    AWS + Kubernetes + Microservicios
    ↓ Infraestructura productiva real a gran escala
```
